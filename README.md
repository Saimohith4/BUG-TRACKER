A full-stack issue tracking system that allows teams to manage projects, create tickets, assign tasks, and track progress using a Kanban board.

This project demonstrates REST API development using FastAPI and a modern frontend using Vite.

---

## Features

### Authentication
- User registration
- User login
- Logout functionality

### Project Management
- Create projects
- View project list
- Assign tickets to projects

### Ticket Management
- Create tickets
- Assign user to ticket
- Update ticket status
- View ticket details

### Kanban Board
- To Do column
- In Progress column
- Done column
- Drag-and-drop ticket movement

### Dashboard
- Sidebar navigation
- Ticket metrics overview
- Ticket details modal
